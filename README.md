# AlwaysArrowMap
## English

Always Arrow Map is a client-side Fabric mod that improves Minecraft maps by always showing the player arrow, even when the player is outside the map area.

Off-map player markers are automatically converted into normal player arrows and correctly rotated based on the player’s view direction. Map tracking is set to unlimited, so the arrow remains visible at all times.

The mod also fixes dimension-related issues by ensuring consistent behavior between dimensions such as the Nether and Overworld.

This mod is lightweight, fully client-side, and does not require the server to be installed.

---

## Deutsch

Always Arrow Map ist eine clientseitige Fabric-Mod, die Minecraft-Karten verbessert, indem der Spielerpfeil immer angezeigt wird – selbst wenn sich der Spieler außerhalb des Kartenbereichs befindet.

Off-Map-Spielermarkierungen werden automatisch in normale Spielerpfeile umgewandelt und korrekt an die Blickrichtung des Spielers angepasst. Zusätzlich wird unbegrenztes Tracking aktiviert, sodass der Pfeil jederzeit sichtbar bleibt.

Die Mod behebt außerdem Dimensionsprobleme und sorgt für ein konsistentes Verhalten zwischen Dimensionen wie Nether und Overworld.

Die Mod ist leichtgewichtig, vollständig clientseitig und benötigt keine Installation auf dem Server.
